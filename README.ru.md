# ai-limits

[English](README.md) | Русский

Простой просмотр лимитов AI-подписок. Codex, Claude, Cursor.

`ai-limits` — локальный инструмент для просмотра доступных данных об использовании и лимитах поддерживаемых AI-провайдеров. Его можно использовать как десктопное приложение или через терминал.

## Интерфейсы

### Десктопное приложение

Десктопное приложение сейчас находится в beta-статусе.

- macOS уже работает как приложение.
- Windows и Linux собраны и проходят тестирование у живых пользователей.
- Интерфейс функциональный, но пока ранний.

Скачать: https://github.com/md2it/ai-limits/releases

Локальный запуск десктопного приложения в режиме разработки:

```sh
npm run tauri:dev
```

### Терминальный UI

Терминальный интерфейс остаётся доступен.

Запуск из репозитория:

```sh
./bin/ai-limits
```

Справка:

```sh
./bin/ai-limits --help
```

Подробнее о терминальном UI: [docs/terminal-ui.md](docs/terminal-ui.md).

## Что показывает

- Текущие лимиты Codex, Claude и Cursor, когда данные доступны.
- Информацию об использовании, если поддерживаемый источник может её предоставить.
- Данные из локальных файлов, CLI провайдеров и других поддерживаемых источников.
- Стандартный, структурированный и raw-вывод в терминале.
- Повторные проверки через terminal watch mode.

## Текущие ограничения

- Пока нет установщика DMG для macOS.
- macOS-сборки подписаны и нотаризованы; Windows и Linux пока без подписи.
- Десктопные уведомления сейчас работают только на macOS.
- Некоторые локальные источники данных Codex и Claude могут пока не работать на Windows и Linux.
- Источники данных через CLI ожидаются как наиболее переносимый вариант между платформами.
